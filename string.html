이 문서는 C++에서 많이 쓰는 std::string 함수들과 lambda, 그리고 정렬용 비교자 std::greater<> 를 한 번에 정리한 치트시트 형태의 HTML입니다.

PART 1

std::string 기본 & 자주 쓰는 함수

1.1 문자열 길이 구하기

함수 설명 예시
size() 문자열 길이 반환 str.size()
length() size()와 동일 str.length() 
// 길이 구하기 예시
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "Hello World";
    cout << s.size() << "\n";    // 11
    cout << s.length() << "\n";  // 11
}

1.2 문자열 합치기 (연결)

#include <iostream>
#include <string>
using namespace std;

int main() {
    string a = "Hello";
    string b = "World";

    string c = a + b;   // "HelloWorld"
    a += "!!!";         // "Hello!!!"

    cout << c << "\n";
    cout << a << "\n";
}

1.3 인덱스로 문자 접근

  • str[i] : 단순 인덱스 (범위 검사 X)
  • str.at(i) : 범위 검사 O, 잘못된 인덱스면 예외 발생
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "abcde";
    cout << s[0] << "\n";     // 'a'
    cout << s.at(1) << "\n";   // 'b'
}

1.4 부분 문자열 뽑기: substr()

substr(시작인덱스, 길이)

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "abcdef";
    string t = s.substr(2, 3);  // 인덱스 2부터 3글자 → "cde"
    cout << t << "\n";
}

1.5 문자열 검색: find()

find("패턴")은 패턴이 시작되는 인덱스를 반환하고, 없으면 string::npos를 반환한다. 

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "hello world";
    size_t pos = s.find("wor");

    if (pos != string::npos) {
        cout << "찾음! 위치 = " << pos << "\n"; // 6
    } else {
        cout << "못 찾음\n";
    }
}

1.6 치환 / 삭제 / 삽입: replace(), erase(), insert()

#include <iostream>
#include <string>
using namespace std;

int main() {
    string s = "abcdef";

    // replace(시작, 길이, "새 문자열")
    s.replace(1, 3, "X");   // a + X + ef  → "aXef"
    cout << s << "\n";

    // erase(시작, 길이)
    s = "abcdef";
    s.erase(2, 3);          // ab + f → "abf"
    cout << s << "\n";

    // insert(위치, "삽입할 문자열")
    s = "Hello";
    s.insert(1, "!!!");     // H + !!! + ello → "H!!!ello"
    cout << s << "\n";
}

1.7 문자열 정렬하기

#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string s = "dcba";

    sort(s.begin(), s.end());            // 오름차순: "abcd"
    cout << s << "\n";

    sort(s.begin(), s.end(), greater<char>()); // 내림차순: "dcba"
    cout << s << "\n";
}

참고: greater<char>() 를 쓰면 문자 기준 “큰 순서” = 내림차순.

PART 2

C++ 람다(lambda) 함수

2.1 람다 기본 형태

[캡처] (매개변수) -> 반환타입 {
    // 코드
}
#include <iostream>
using namespace std;

int main() {
    // x를 받아서 x*x를 반환하는 람다
    auto square = [](int x) {
        return x * x;
    };

    cout << square(5) << "\n";  // 25
}

2.2 sort에서 람다로 정렬 기준 주기

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> v = {5, 1, 4, 2, 3};

    // 내림차순 정렬 (람다 사용)
    sort(v.begin(), v.end(), [](int a, int b) {
        return a > b;   // a가 더 크면 앞으로
    });

    for (int x : v) cout << x << " ";  // 5 4 3 2 1
    cout << "\n";
}

2.3 캡처 사용 예시

[k]는 바깥 변수 k를 람다 안으로 복사해서 쓰겠다는 의미.

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> v = {10, 11, 12, 13, 14};
    int k = 5;

    // k로 나눈 나머지 기준으로 정렬
    sort(v.begin(), v.end(), [k](int a, int b) {
        return (a % k) < (b % k);
    });

    for (int x : v) cout << x << " ";
    cout << "\n";
}

캡처 문법: []는 아무것도 캡처하지 않음, [&]는 외부 변수를 참조로 캡처, [=]는 값으로 캡처 등.

PART 3

std::greater<> 로 내림차순 정렬하기

3.1 greater<> 란?

std::greater<T> 는 사실상 bool operator()(T a, T b) { return a > b; } 를 가지는 “함수 객체(functor)”이다. 즉, 정렬 기준을 < 대신 >로 바꿔주는 역할을 한다.

3.2 vector<int> 에 사용

#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> v = {5, 1, 4, 2, 3};

    sort(v.begin(), v.end(), greater<int>()); // 내림차순

    for (int x : v) cout << x << " ";  // 5 4 3 2 1
    cout << "\n";
}

3.3 문자열에도 사용 가능

#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

int main() {
    string s = "dcba";

    sort(s.begin(), s.end(), greater<char>());
    cout << s << "\n";  // "dcba" (내림차순)

    return 0;
}

3.4 greater vs 람다 비교

방법 코드 특징
greater<> sort(v.begin(), v.end(), greater<int>()); 숫자/문자 단순 내림차순 때 간단
lambda sort(v.begin(), v.end(), [](int a,int b){ return a > b; }); 조건을 자유롭게 커스터마이징 가능

단순 내림차순이면 greater<>가 더 짧고, 복잡한 정렬 기준(예: 길이 우선, 그 다음 사전순 등)이 필요하면 람다를 쓰는 것이 일반적이다.
