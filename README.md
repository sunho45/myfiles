# My Files Repository

이 저장소는 제가 공부하면서 만든 코드, 정리 자료, 이미지 등을 모아두는 공간입니다.  
처음 GitHub를 배우는 단계라서 간단한 파일부터 차근차근 업로드하고 있습니다.

---

## 📁 구성 안내

- **C++ 연습 코드**  
  알고리즘 문제, 자료구조 구현, 간단한 테스트 코드 등이 포함되어 있습니다.

- **HTML/CSS/JS 파일**  
  웹 프론트엔드 연습용 페이지와 실험용 코드가 들어 있습니다.

- **Python 코드**  
  간단한 스크립트나 테스트용 파이썬 파일이 포함되어 있습니다.

- **문서 / 이미지 / PDF**  
  공부 과정에서 참고한 자료나 정리한 파일들입니다.

---

## 🛠 사용한 기술들

- Git / GitHub 기본 기능
- C++ (기본 문법, 알고리즘)
- Python (간단한 스크립트)
- HTML / CSS (웹 기본)
- Linux 명령어 및 환경

---

## ✅ Git 사용 기록

이 저장소는 Git을 처음 배우며 다음 과정을 연습하기 위해 사용합니다.

- Git 저장소 초기화 (`git init`)
- 파일 추가 및 커밋 (`git add`, `git commit`)
- 원격 저장소 연결 (`git remote add origin`)
- GitHub로 업로드 (`git push`)
- 브랜치 관리(`master` → `main` 등)
